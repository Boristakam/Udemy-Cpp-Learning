by default data is passed to functions by value (making a copy of the variables passed)

we saw that with arrays we are passing the address so arr elements can therefore be changed in the function's body
    what if we want to do it with other variable types ( this is where passing by reference comes in using the & in param list)
    the function parameter (formal parameter) will now be an alias to the actual parameter


passing by reference means we dont make a copy of the data which could be large and therefore we save time 

Note: the parameters will now need to be const if you do not want them modified by accident or otherwise no const
