class attributes and methods are members of that class 
by default all members of a class are private so we make them public to access them in the program 
we need objects to access members of a class 
can do so using the dot or arrow operator (member of pointer operator) 
    e.g objectName.membername(init val)
        objectName.membername
        objectName->membername(init val) //not applicable here but important to know that the arrow dereferences the pointer for you
        
        Note: if you have a pointer to an object (or class object), 
        you need to dereference it first to get to the object then you can use the dot or arrow operator
        e.g (*objectName).membername(init val) //use bracket because dot operator has a higher precedence that the *
        

remember some class members may not be accessibily if desingned not to be (more on it later)

Use assignement operation '=' instead of curly brackets to modify or initialise members outside of the class definition
