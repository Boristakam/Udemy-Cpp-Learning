this is how they're overloaded (as non-member functions) starting with the insersion operator:
the first arg is an output stream obj of type ostream the 2nd is a ref the obj whose data we want to display (insert in the output stream)
std::ostream &operator<<(std::ostream &os, const Mystring &obj){ //notice the ref is constant 
    os << obj.str;  //if friend function required
    os << obj.get_str //if not friend function required and you need to access the getter function with an object
    return os; 
}
