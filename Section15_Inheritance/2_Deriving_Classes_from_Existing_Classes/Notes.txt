class Base {
    // Base class members...
}

class Derived_class: access-specifier Base {
    // Derived_class members...
}

access specifier can be public, private, or protected, if you dont provide it, it is set to access private members 

public inheritance is the most common: established is-a relationship

private & protected inheritance: establishes has-a relationship (the Derived class has-a Base class). is emplimented in terms of
    relationship and is different to composition
