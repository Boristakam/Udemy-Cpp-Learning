As seen before a derived class can directly/explicitly invoke Base class methods 
It can also override or redefine them 
very powerful for polymorphism (in the next section)

Static Binding of Method calls:
    Means that the compiler determines what methods to call based on what it sees at compile time 
    Default binding in C++ is static binding
    Derived class objects will be Derived::mehod_name 
    But we can explicitly invoke Base::method_name from Derived::method_name (so withing the derived class use a Base object to call a Base method)
    Ok but limited, polymorphism is more dynamic and powerful
