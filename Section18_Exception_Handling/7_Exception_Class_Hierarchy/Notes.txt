c++ provides exception class hierarchy 
    std::exception is the base class. it has other exception classes derived from it such as bad_alloc, system_error, domain_error etc
    all subclasses implement the what() virtual function 
    we can create our own user-defined exception hierarchy

    virtual const char *what() const noexcept; // what() virtual function displays whatever msg we want 


    deriving our class from std::exception

    class IllegalBalanceException: public std::exception
    {
    public:
        IllegalBalanceException() noexcept = default;   //noexcept means compiler will no throw an exception
        ~IllegalBalanceException() = default;
        virtual const char* what() const noexcept
        {
            return "Illegal ballance exception";
        }   
    }
