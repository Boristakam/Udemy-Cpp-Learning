variables are assigned individual addresses in RAM. in other words they're an abstraction for a memory location
they have 2 main properties, first is their type and their value which may change as the program executes

we can create our own types to define variable. thanks to c++ being an object oriented language i.e 'person Boris' 
where person is the type and Boris is the variable 

keywords cannot be used as variable names like return, it cannot start with a number, it is case senstive 
so int Age is different to int age, cannot have operators like age+1, no special characters 

never use a variable before initialising them, it is best to declare them as close to where you need to use them 

be consistent with naming scheme (usually specific for companies in the industry)
